## Install instructions:

 1. Add NAudio & Newtonsoft dependencies through NuGet Packages if needed
 1. Build Project
 1. Extract "assets.rar" to game folder (usually "Debug" or "Release")
 1. Install "Diogenes" font inside "fonts" directory
 1. Run!
